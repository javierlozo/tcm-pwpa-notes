# 03 · Injection Attacks

The biggest section in the course and where most of the classic web bugs live.

## Topics from the course

- LFI / RFI (Local and Remote File Inclusion)
- SQL Injection (basic, blind, second-order)
- XSS (reflected, stored)
- Command Injection (basic, blind)
- SSTI (Server-Side Template Injection)
- XXE (XML External Entity)
- Insecure File Uploads (client-side bypass, server-side bypass)

Each gets its own file with payloads I've actually used in labs and how to fix the underlying issue.
