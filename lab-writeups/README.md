# Lab Writeups

Walkthroughs of labs I've completed in **authorized environments only**:

- TCM PWPA course labs
- PortSwigger Web Security Academy
- HackTheBox web challenges
- My own homelab targets

## Rules

- **No real client work.** Anything from paid engagements stays out of this repo.
- **No embargoed bug bounty findings.** Wait until disclosure.
- **Sanitize.** Strip session tokens, internal hostnames, customer data from all artifacts.
- **Filename pattern:** `YYYY-MM-DD-platform-lab-name.md` — e.g. `2026-05-12-portswigger-blind-sqli.md`.

See `_template.md` in this folder when you start a new writeup.
