# TCM PWPA — Study Notes

Personal study notes for the **[TCM Security Practical Web Pentest Associate (PWPA)](https://academy.tcm-sec.com/p/practical-web-pentest-associate)** certification.

Written in my own words while working through the course. Public so other learners can use them, and so my prep is dated and visible.

— Luis Javier Lozoya · [luislozoya.com](https://www.luislozoya.com) · [github.com/Javierlozo](https://github.com/Javierlozo)

## Structure

```
00-setup/              Lab environment, Burp setup, browser config
01-web-fundamentals/   HTTP, sessions, auth, cookies, same-origin
02-burp-suite/         Proxy, Repeater, Intruder, Scanner, extensions
03-owasp-top-10/       Per-category notes with payloads + fixes
04-recon/              Passive + active recon checklists
05-methodology/        Engagement workflow, reporting templates
lab-writeups/          Per-lab walkthroughs (lab environments only)
```

Each folder has its own `README.md` as a topic index.

## How I'm using this repo

- **Daily commits** while studying. Small commits beat polished ones.
- **Notes in my own words** — not transcripts of TCM lecture content.
- **Lab writeups for sanctioned environments only** (TCM labs, PortSwigger Academy, HackTheBox, my own homelab). No client work, no embargoed bug bounty findings.

## Disclaimer

These are personal study notes. Not affiliated with TCM Security or any course provider. All techniques described here are applied **only in authorized lab environments** I own or have explicit permission to test.

Course content is paraphrased and reorganized — refer to the original [PWPA course](https://academy.tcm-sec.com/p/practical-web-pentest-associate) for the canonical material.
